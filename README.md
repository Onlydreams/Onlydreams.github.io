## 本地开发

Ruby 版本以 [`.ruby-version`](.ruby-version) 为准。仓库操作统一使用项目脚本。

macOS / Linux：

```bash
bash bin/setup
bash bin/test
bash bin/serve
```

Windows PowerShell：

```powershell
.\bin\setup.ps1
.\bin\test.ps1
.\bin\serve.ps1
```

完整的开发、内容与验证规范见 [`AGENTS.md`](AGENTS.md)。
